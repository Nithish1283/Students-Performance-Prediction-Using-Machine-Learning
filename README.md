## 📌 Project Overview

The **Students Performance Prediction Using Machine Learning** project predicts a student's academic performance using various demographic and educational factors. The project applies machine learning techniques to analyze the dataset, identify patterns, and generate accurate predictions that can help educators and students understand performance trends.

This project demonstrates the complete Machine Learning workflow including:

- Data Collection
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Model Training
- Model Evaluation
- Prediction
